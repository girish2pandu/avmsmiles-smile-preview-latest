How to get a free HuggingFace token (2 minutes, no bank account):
  1. Go to huggingface.co/join — sign up free with email
  2. Click your profile photo → Settings → Access Tokens
  3. Click "New token", name it anything (e.g. "avmsmiles"), select Read role
  4. Click Create token, then copy it (starts with hf_)
  5. Paste it below and test
